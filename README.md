Time Converter (C++)
📌 Project Description

Time Converter is a simple C++ project developed using Object-Oriented Programming (OOP) concepts. This project allows users to convert time from one format to another using a menu-driven program.

The project provides two conversion options:

Convert Total Seconds into HH:MM:SS format.
Convert HH:MM:SS format into Total Seconds.

This project helps in understanding Classes, Objects, Functions, Switch Case, User Input, and Basic Arithmetic Operations in C++.

✨ Features
Convert Seconds to HH:MM:SS
Convert HH:MM:SS to Seconds

🚀 How to Run
Open the project in VS Code.
Compile the program:
g++ Time-converter.cpp
Run the program:
./a.exe
Select the conversion option and enter the required values.
